# FLEXT Target Oracle OIC

Singer Target para enviar dados a recursos Oracle Integration Cloud.

Descricao oficial atual: "FLEXT Target Oracle OIC - Singer Target for Oracle Integration Cloud".

## O que este projeto entrega

- Publica dados de pipeline para destinos OIC.
- Padroniza fluxo de carga para integracao cloud Oracle.
- Apoia cenarios de entrega de dados para processos OIC.

## Contexto operacional

- Entrada: stream Singer com dados preparados.
- Saida: recursos OIC atualizados via destino configurado.
- Dependencias: acesso Oracle OIC e configuracao de destino.

## Estado atual e risco de adocao

- Qualidade: **Alpha**
- Uso recomendado: **Nao produtivo**
- Nivel de estabilidade: em maturacao funcional e tecnica, sujeito a mudancas de contrato sem garantia de retrocompatibilidade.

## Diretriz para uso nesta fase

Aplicar este projeto somente em desenvolvimento, prova de conceito e homologacao controlada, com expectativa de ajustes frequentes ate maturidade de release.
